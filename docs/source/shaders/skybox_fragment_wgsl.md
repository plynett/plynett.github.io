# `shaders/skybox.fragment.wgsl`

[Source](../../../shaders/skybox.fragment.wgsl)

## What This Shader Does

Samples the skybox cube texture using the direction generated by `skybox.vertex.wgsl`.

## Binding Contract

- `1`: cube texture.
- `2`: sampler.

Binding `0` is the inverse view-projection uniform consumed by the vertex shader in the same bind group.

## Change Notes

This shader is deliberately small. Environment appearance should usually be changed by replacing cube-map textures rather than editing shader logic.
